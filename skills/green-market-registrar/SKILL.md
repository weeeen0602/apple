# Skill: Green Market Registrar (V3.0)

## 🌟 技能描述
**Green Market Registrar** 是一個專為「綠生活循環市集」設計的自動化報告生成工具。它能將非結構化的文字紀錄（如學生攤位分工、環保行動、挑戰與收穫）進行解析，並自動判定「綠色價值 (Green Status)」，最後產出結構化、視覺化的 **PDF 專業活動報告**。

## 🚀 核心功能
- **數據解析**：使用 Regex 從 `raw_data.txt` 中提取班級、姓名、攤位名稱、任務、挑戰與收穫。
- **價值判定 (Green Status)**：透過關鍵字比對（如：回收、改造、循環、二手），自動為攤位標記 ✅ (Green)、⚠️ (Warning) 或 ❌ (Non-Green)。
- **視覺化排版**：自動生成包含 Header、Body (資訊卡片)、Visuals (影像描述佔位符) 與 Footer 的 PDF 文檔。
- **品質檢查**：自動檢查 PDF 檔案大小，確保不超過 4MB 的傳輸限制。

## 📋 使用說明

### 1. 準備輸入資料 (`references/raw_data.txt`)
請確保輸入檔案遵循以下格式（每組攤位資訊應清晰分隔）：
```text
班級: 61班 | 座模: 1 | 姓名: 王小明 | 攤位名稱: 舊物重生站
任務: 將舊衣改造為環保袋
挑戰: 縫紉技術不夠熟練
收穫: 學會了基礎縫技巧
---
班級: 62班 | 座模: 2 | 姓名: 李小華 | 攤位名稱: 綠色回收站
任務: 收集廢棄塑膠瓶
挑戰: 無
收穫: 了解塑膠分類
```

### 2. 執行指令
在 PowerShell 環境下，切換至技能目錄並執行 Python 腳本：
```powershell
cd skills\green-market-registrar
uv run python scripts\parser.py
```

### 3. 查看成果
執行成功後，請至 `output\` 目錄查看生成的 `report.pdf`。

## 📊 數據 Schema (五維度)
| 維度 | 欄位名稱 | 說明 |
| :--- | :--- | :--- |
| **Basic Info** | 身份識別 | 班級、座模、姓名、攤位名稱 |
| **Task** | 投入內容 | 學生參與的環保行動內容 |
| **Green Status** | 價值判定 | 透過關鍵字自動判定 (✅/❌/⚠️) |
| **Challenges** | 過程挑戰 | 執行過程中的障礙與困難 |
| **Achievements**| 產出收穫 | 學生學到的知識或達成的目標 |

## ⚠️ 注意事項與限制
- **字型限制**：目前版本 PDF 僅支援英文與數字標題，中文字元可能無法正確渲染（後續版本將整合 .ttf 字型）。
- **檔案大小**：產出的 PDF 應維持在 **4MB 以下**。
- **依賴套件**：執行前請確保已安裝 `reportlab`。
