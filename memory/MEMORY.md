# 技能開發專案：green-market-registrar

## 🛠️ 技能核心定義
這是一個自動化的「活動成果報告生成器」，旨在將非結構化的文字與影像描述，轉換為專業、結構化且符合規格（< 4MB）的 PDF 報告。

### 📊 數據維度 (Data Schema)
開發過程中定義了五維度的數據結構：
1. **身份 (Who)**：班級、座號、姓名、攤位名稱。
2. **行動 (What/Green Status)**：分工內容，並透過「綠色行動關鍵字」自動判定環保貢獻度 (✅/❌/⚠️)。
3. **障礙 (Challenges)**：執行過程中的困難與挑戰。
4. **成果 (Achievements)**：學習產出與達成目標。
5. **視覺 (Visuals)**：影像描述與照片的邏輯配對。

## ⚙️ 技術規格與防護機制
- **解析層 (Parsing)**：使用 Regex 提取欄位，並透過關鍵字比對引擎進行價值判定。
- **渲染層 (Rendering)**：建立內部的「排版清單 (Layout List)」，實現文字與影像描述的邏輯配對。
- **驗證層 (Validation)**：實作 `Size Constraint Check`，確保產出的 PDF 檔案大小符合 4MB 的部署限制，必要時執行自動優化策略。
